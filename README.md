## WebStack - 个人网址导航

<img src="./assets/images/logo@2x.png" width="100" alt="WebStack Logo">

一个基于 [WebStackPage](https://github.com/WebStackPage/WebStackPage.github.io) 的极简、高效个人网址导航页。

[![Demo](https://img.shields.io/badge/Demo-nav.555212.xyz-blue?style=flat-square&logo=google-chrome)](http://nav.555212.xyz)
![License](https://img.shields.io/badge/License-MIT-green?style=flat-square)

---


### 1. 数据结构与逻辑优化
**数据驱动**: 引入 `nav-data.js` 与 `nav-renderer.js`，将数据与视图解耦，只需编辑 JSON 即可管理导航。

### 2. 性能优化
**去依赖**: 移除 Google Fonts、Google Analytics 及广告代码。
**本地化**: 核心资源本地化部署，提升加载速度。

### 3. 品牌与信息更新
**定位**: 转型为通用“个人网址导航”。
**地址**: 更新为 `nav.555212.xyz`。
**SEO**: 优化 Title、Description 与 Keywords。
